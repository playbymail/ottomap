= OttoMap CLI Commands

== Global Flags

These flags are available on all commands:

* `--debug`: Log debugging information.
* `--log-file <path>`: Set log file path.
* `--log-with-default-flags`: Log with default flags.
* `--log-with-shortfile`: Log with short file name (default: true).
* `--log-with-timestamp`: Log with timestamp.
* `--quiet`: Log less information.
* `--verbose`: Log more information.
* `--show-version`: Show version.

== render

The primary command. Loads and parses turn report files from the `data/input` directory and generates a Worldographer `.wxx` map file in `data/output`.

----
ottomap render --clan-id 0138
----

=== Flags

* `--clan-id <id>`: Clan for output file names (required).
* `--data <path>`: Path to root of data files (default: `data`).
* `--max-turn <yyyy-mm>`: Last turn to map. Limits map generation to turns up to and including this date.
* `--auto-eol`: Automatically convert line endings (default: true).
* `--origin-grid <id>`: Grid ID to substitute for `##` in obscured starting hexes.
* `--save-with-turn-id`: Add turn ID to output file name.
* `--shift-map`: Shift map up and left to remove empty space (default: true).
* `--show-grid-coords`: Show grid coordinates (XX CCRR) on the map.
* `--show-grid-numbers`: Show grid numbers (CCRR) on the map.
* `--show-origin`: Show origin hex on the map.
* `--solo`: Limit parsing to a single clan.
* `--solo-element <id>`: Limit parsing to a single element of a clan.
* `--top-left <coord>`: Experimental: top left corner of rendered map.
* `--bottom-right <coord>`: Experimental: bottom right corner of rendered map.
* `--warn-on-invalid-grid`: Warn on invalid grid ID (default: true).
* `--warn-on-new-settlement`: Warn on new settlement (default: true).
* `--warn-on-terrain-change`: Warn when terrain changes (default: true).

=== Examples

Render all turns for clan 0138:

----
ottomap render --clan-id 0138
----

Render up to a specific turn:

----
ottomap render --clan-id 0138 --max-turn 0901-12
----

Render with grid coordinates displayed:

----
ottomap render --clan-id 0138 --show-grid-coords
----

== version

Displays the current version of OttoMap.

----
ottomap version
----

== list

List information about available turn data.

=== list clans

Lists all clans found in the data directory.

----
ottomap list clans
----

=== list turns

Lists all turns found in the data directory.

----
ottomap list turns
----

== dump

Dump internal data to a file. This is a diagnostic command used for debugging.

----
ottomap dump
----
