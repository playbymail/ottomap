Map faster! Map differently!

Everything you need to create maps from your turn reports

OttoMap is an open-source web application that allows you to create maps from your turn reports. It is designed to be used with the TribeNet turn reporting system (but can be used with any turn reporting system that just happens to have the same format).

Secure by default

You own your data. Delete it at any time. Reports and maps are password protected and are never shared with anyone.

Learn more →

Upload turn reports, download maps

Save your turn reports as text files and upload them to the server. The files are read and any errors are reported back to you. Maps are generated automatically and can be downloaded as Worldographer files.

Learn more →

Community supported

OttoMap questions can be asked and answered on the #mapping-tools channel in the TribeNet Discord server.
